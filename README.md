##  Descripción del proyecto
Este proyecto consiste en la creación de una página web BASICA personal utilizando HTML.  
En ella se incluye información básica sobre mí, mis gustos, intereses y metas.  

El objetivo es aprender a utilizar GitHub como herramienta de control de versiones, registrando cambios mediante commits a lo largo del desarrollo del proyecto.
